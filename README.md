A Swift Package Manager project scaffold for the `SpecificationConfig` library. The package currently exposes a placeholder type so you can start adding functionality and tests right away.

## Requirements

- Swift 6.0 or later
- Xcode 16.2 (Xcode 26.0) or later (for development on macOS)

## Building and Testing

Clone the repository and use Swift Package Manager:

```bash
swift build
swift test
```

You can also open the package in Xcode:

```bash
open Package.swift
```
